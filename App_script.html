Google World Setup

This guide explains, step by step, how to connect the Rally App to Google World

Before You Start

You need a Google account to continue.

If you do not already have one, you will create one in the next step.

• If you already use Gmail, YouTube, or Google Drive, you already have one.

• If not you will be guided through creating a free Google account.

Step 1 – Sign in or create a Google Account

If Google Drive opens, you are already signed in.

If you see a sign-in page:

  1. Click Create account.
  2. Choose For myself.
  3. Follow the on-screen instructions.
  4. When finished, return to drive.google.com.

Step 2 – Open Google Drive

There are two ways to open Google Drive.

Option A – Direct link

  1. Open your web browser.
  2. Type drive.google.com and press Enter.

Option B – Using the Google apps menu (9 dots)

  1. Open a new browser tab, click the 9 dots near the top right-hand corner of the page.
  2. Select Drive from the options presented.
Step 3 – Create a New Apps Script Project
  1. In Google Drive, click (+New) top-left under the Drive logo.
  2. Click More, then if App Script can’t be seen select More at the bottom of the options.
  3. Click Google Apps Script.
  4. Select Create script.

Step 4 – Prepare the Apps Script Editor (Important)

When the Apps Script editor opens, you will see:

  1. A file named Code.gs will be highlighted in the left panel.
  2. Some initial sample code already in the editor pane to the right.

Important: Do NOT delete this code yet.

Step 5 – Copy Rally Code.gs from Admin
  1. Go back to the Rally Admin page.
  2. Select the Entries button, then click the Copy Code.gs button.
  3. When the page is presented, click inside the page
  4. Press Ctrl+A (Windows) or Cmd+A (Mac) to select all text.

Step 6 – Replace the Default Script

  1. Return to the Google Apps Script browser tab.
  2. Click anywhere inside the code editor.
  3. Press Ctrl+A (Windows) or Cmd+A (Mac) to select all text.
  4. Press Delete to remove the sample code.
  5. Press Ctrl+V (Windows) or Cmd+V (Mac) to paste the Rally Code.gs script. You should now see a large block of Rally code in the editor.

Step 7 – Save the Script

  1. Look above the editor for the Save icon (disk symbol).
  2. Click Save, or press Ctrl+S / Cmd+S.
  3. Wait until the saving message disappears.

Step 8 – Deploy as a Web App

  1. Click Deploy (top-right).
  2. Click New deployment.
  3. In the top left of the card, click the gear icon to the right of Select type.
  4. Select Web app from the options presented.
  5. For Execute as, select “Me” (your email address).
  6. Set Who has access to Anyone (or Anyone with the link).
  7. Click Deploy.
  8. When prompted, select Authorize access. If warned, click Advanced, then select Go to project (unsafe).
  9. When new details are presented, tick Select all, then select Continue.

Step 9 – Copy the Web App URL

  1. After deployment, Google shows a Web app URL.
  2. Copy this URL.

Step 10 – Connect from Rally Admin

  1. Return to the Rally Admin page.
  2. Select Entries then paste the Web app URL into the URL field.
  3. Enter your mobile number, including international prefix (for example +27).
  4. Click Save & test.
  5. The message at the top of the card should now indicate “Connected”.
  6. Your Google World setup is now complete and standing by for any Rally you work on.
  7. The setup of Google World is now complete and no further interaction directly with Google Drive is required for current or future rally projects.

Note: After your first upload of Admin data a folder will be created in Google My Drive titled Time and Distance Rallies.

Using Google World

  1. Upload Entries prepares your Google entry form, including Speed Group options ready to be shared.
  2. Download Entries automatically populates the entries table.
  3. Using the Controls page and selecting its Controls data button opens the interface card so you can upload STC and Marshal data, preparing response forms ready to share.
  4. Download STC and Marshal responses automatically populate the controls page in preparation for review and printing of results.
